photogenix by janpaul123

Photoframe project in PHP/Javascript

Photogenix is a script to run full-screen in a browser on a photoframe, television set or whatever. It displays your photo albums, RSS feeds, traffic information, a clock, and sure much more to come. All with very slick animations.
